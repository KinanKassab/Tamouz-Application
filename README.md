# كشافة تموز - تطبيق الفوج الأول

## نظرة عامة

كشافة تموز هو تطبيق ويب حديث ومتطور مصمم خصيصاً لإدارة أنشطة الفوج الكشفي الأول. يوفر التطبيق منصة شاملة للتواصل بين أعضاء الفوج، تتبع الأنشطة والفعاليات، تطوير المهارات الكشفية، وبناء مجتمع كشفي قوي ومترابط.

## الميزات الرئيسية

### 🔐 نظام المصادقة المتقدم
- تسجيل دخول آمن باستخدام Supabase Authentication
- إنشاء حسابات جديدة مع التحقق من البريد الإلكتروني
- إعادة تعيين كلمة المرور
- إدارة الجلسات والأمان

### 🏠 الصفحة الرئيسية التفاعلية
- لوحة معلومات شخصية تعرض إحصائيات المستخدم
- إجراءات سريعة للوصول للميزات الأساسية
- عرض الأنشطة الحديثة والقادمة
- إشعارات فورية للأحداث المهمة

### 📅 إدارة الأنشطة والفعاليات
- عرض جدول الأنشطة بتصميم جذاب وسهل القراءة
- تصفية الأنشطة حسب الحالة (قادمة، مسجل بها، مكتملة)
- تسجيل الحضور في الأنشطة
- عرض تفاصيل شاملة لكل نشاط

### 🧭 دليل المهارات الكشفية
- مكتبة شاملة للمهارات الكشفية مقسمة حسب الفئات
- تتبع تقدم المستخدم في كل مهارة
- نظام تقييم بالنجوم لصعوبة المهارات
- بحث متقدم في المهارات

### 👥 مجتمع الكشافة
- منصة تواصل اجتماعي للأعضاء
- نشر المحتوى والتفاعل مع المنشورات
- عرض قائمة الأعضاء والمجموعات
- نظام الإعجاب والتعليقات

### 👤 الملف الشخصي المتكامل
- عرض وتعديل المعلومات الشخصية
- تتبع الشارات المكتسبة والإنجازات
- سجل الأنشطة المشارك بها
- إحصائيات شخصية مفصلة

### 📱 تطبيق ويب تقدمي (PWA)
- يعمل بدون اتصال إنترنت
- قابل للتثبيت على الأجهزة المحمولة
- إشعارات فورية
- تحديثات تلقائية في الخلفية

## الأهمية والاستخدامات

### للقادة الكشفيين:
- **إدارة فعالة**: تنظيم الأنشطة وتتبع حضور الأعضاء
- **التواصل المباشر**: إرسال الإشعارات والتحديثات للأعضاء
- **تقييم التقدم**: مراقبة تطور مهارات الأعضاء وإنجازاتهم
- **التخطيط**: جدولة الأنشطة والفعاليات بسهولة

### للأعضاء:
- **التعلم التفاعلي**: الوصول لدليل شامل للمهارات الكشفية
- **التواصل الاجتماعي**: بناء علاقات قوية مع أعضاء الفوج
- **تتبع الإنجازات**: مراقبة التقدم الشخصي والشارات المكتسبة
- **سهولة المشاركة**: التسجيل في الأنشطة ومتابعة الجدول

### للمؤسسات الكشفية:
- **الرقمنة**: تحويل العمليات التقليدية إلى نظام رقمي حديث
- **الكفاءة**: تحسين إدارة الموارد والوقت
- **الشفافية**: توثيق جميع الأنشطة والإنجازات
- **النمو**: جذب أعضاء جدد من خلال التقنيات الحديثة

## التقنيات المستخدمة

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Supabase (PostgreSQL, Authentication, Real-time)
- **UI Framework**: Lucide Icons للأيقونات
- **PWA**: Service Workers, Web App Manifest
- **Build Tool**: Vite
- **Styling**: CSS Custom Properties, Flexbox, Grid

## المطور والمساهمون

**المطور الرئيسي**: كنان قصاب (Kinan Kassab)
- مطور ويب متخصص في التطبيقات التفاعلية
- خبرة في تطوير تطبيقات الويب التقدمية
- شغوف بتطوير حلول تقنية للمجتمعات الشبابية

## التثبيت والتشغيل

```bash

npm install

# تشغيل الخادم المحلي
npm run dev

# بناء المشروع للإنتاج
npm run build
```

## إعداد قاعدة البيانات

1. إنشاء مشروع جديد في [Supabase](https://supabase.com)
2. نسخ متغيرات البيئة من `.env.example` إلى `.env`
3. تشغيل migrations قاعدة البيانات
4. إعداد RLS policies
