# كشافة تموز - تطبيق الفوج الأول

## نظرة عامة

كشافة تموز هو تطبيق ويب تقدمي (PWA) حديث ومتطور مصمم خصيصاً لإدارة أنشطة الفوج الكشفي الأول. يوفر التطبيق منصة شاملة للتواصل بين أعضاء الفوج، تتبع الأنشطة والفعاليات، تطوير المهارات الكشفية، وبناء مجتمع كشفي قوي ومترابط.

## الميزات الرئيسية

### 🔐 نظام المصادقة المتقدم
- تسجيل دخول آمن مع واجهة مستخدم حديثة
- إنشاء حسابات جديدة مع التحقق
- إعادة تعيين كلمة المرور
- إدارة الجلسات والأمان
- دعم كامل للغة العربية

### 🏠 الصفحة الرئيسية التفاعلية
- لوحة معلومات شخصية تعرض إحصائيات المستخدم
- إجراءات سريعة للوصول للميزات الأساسية
- عرض الأنشطة الحديثة والقادمة
- إشعارات فورية للأحداث المهمة
- تصميم متجاوب وجذاب

### 📅 إدارة الأنشطة والفعاليات
- عرض جدول الأنشطة بتصميم بطاقات حديث
- تصفية الأنشطة حسب الحالة (قادمة، مسجل بها، مكتملة)
- تسجيل الحضور في الأنشطة بنقرة واحدة
- عرض تفاصيل شاملة لكل نشاط
- مشاركة الأنشطة مع الآخرين

### 🧭 دليل المهارات الكشفية
- مكتبة شاملة للمهارات الكشفية مقسمة حسب الفئات
- تتبع تقدم المستخدم في كل مهارة
- نظام تقييم بالنجوم لصعوبة المهارات
- بحث متقدم في المهارات
- خطوات تعليمية مفصلة لكل مهارة

### 👥 مجتمع الكشافة
- منصة تواصل اجتماعي للأعضاء
- نشر المحتوى والتفاعل مع المنشورات
- عرض قائمة الأعضاء والمجموعات
- نظام الإعجاب والتعليقات
- إنشاء ومشاركة المحتوى

### 👤 الملف الشخصي المتكامل
- عرض وتعديل المعلومات الشخصية
- تتبع الشارات المكتسبة والإنجازات
- سجل الأنشطة المشارك بها
- إحصائيات شخصية مفصلة
- نظام النقاط والمستويات

### 📱 تطبيق ويب تقدمي (PWA)
- يعمل بدون اتصال إنترنت (قيد التطوير)
- قابل للتثبيت على الأجهزة المحمولة
- إشعارات فورية
- تحديثات تلقائية في الخلفية
- أداء سريع ومحسن

### 🎨 تصميم حديث ومتجاوب
- واجهة مستخدم عصرية وجذابة
- دعم الوضع المظلم والفاتح
- تصميم متجاوب لجميع الأجهزة
- انتقالات سلسة وتأثيرات بصرية
- إمكانية الوصول المحسنة

## الأهمية والاستخدامات

### للقادة الكشفيين:
- **إدارة فعالة**: تنظيم الأنشطة وتتبع حضور الأعضاء
- **التواصل المباشر**: إرسال الإشعارات والتحديثات للأعضاء
- **تقييم التقدم**: مراقبة تطور مهارات الأعضاء وإنجازاتهم
- **التخطيط**: جدولة الأنشطة والفعاليات بسهولة

### للأعضاء:
- **التعلم التفاعلي**: الوصول لدليل شامل للمهارات الكشفية
- **التواصل الاجتماعي**: بناء علاقات قوية مع أعضاء الفوج
- **تتبع الإنجازات**: مراقبة التقدم الشخصي والشارات المكتسبة
- **سهولة المشاركة**: التسجيل في الأنشطة ومتابعة الجدول

### للمؤسسات الكشفية:
- **الرقمنة**: تحويل العمليات التقليدية إلى نظام رقمي حديث
- **الكفاءة**: تحسين إدارة الموارد والوقت
- **الشفافية**: توثيق جميع الأنشطة والإنجازات
- **النمو**: جذب أعضاء جدد من خلال التقنيات الحديثة

## التقنيات المستخدمة

### Frontend
- **TypeScript**: لتطوير آمن ومحسن
- **Vite**: أداة بناء سريعة وحديثة
- **CSS Custom Properties**: نظام تصميم موحد
- **Lucide Icons**: مكتبة أيقونات حديثة
- **PWA**: Service Workers و Web App Manifest

### Architecture
- **Clean Architecture**: فصل الاهتمامات وسهولة الصيانة
- **Component-Based**: مكونات قابلة لإعادة الاستخدام
- **State Management**: إدارة حالة مركزية
- **Router System**: نظام توجيه متقدم

### Development Tools
- **ESLint**: فحص جودة الكود
- **Prettier**: تنسيق الكود
- **TypeScript**: فحص الأنواع
- **Vite PWA Plugin**: دعم PWA متقدم

### Design System
- **CSS Variables**: نظام ألوان وتباعد موحد
- **Responsive Design**: تصميم متجاوب
- **Dark Mode**: دعم الوضع المظلم
- **Accessibility**: إمكانية الوصول المحسنة

## البنية المعمارية

```
src/
├── core/                 # النواة الأساسية
│   ├── App.ts           # التطبيق الرئيسي
│   ├── Router.ts        # نظام التوجيه
│   ├── StateManager.ts  # إدارة الحالة
│   ├── AuthManager.ts   # إدارة المصادقة
│   ├── UIManager.ts     # إدارة واجهة المستخدم
│   ├── ThemeManager.ts  # إدارة السمات
│   └── PWAManager.ts    # إدارة PWA
├── pages/               # الصفحات
│   ├── AuthPage.ts      # صفحة المصادقة
│   ├── HomePage.ts      # الصفحة الرئيسية
│   ├── ActivitiesPage.ts # صفحة الأنشطة
│   ├── SkillsPage.ts    # صفحة المهارات
│   ├── CommunityPage.ts # صفحة المجتمع
│   └── ProfilePage.ts   # صفحة الملف الشخصي
├── components/          # المكونات
│   └── BaseComponent.ts # المكون الأساسي
├── services/           # الخدمات
│   └── MockDataService.ts # خدمة البيانات التجريبية
├── types/              # تعريفات الأنواع
│   └── index.ts        # الأنواع الرئيسية
├── styles/             # الأنماط
│   └── main.css        # الأنماط الرئيسية
└── main.ts             # نقطة الدخول
```

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+ 
- npm أو yarn

### خطوات التثبيت

```bash
# استنساخ المشروع
git clone [repository-url]
cd tamouz-scouts-app

# تثبيت المتطلبات
npm install

# تشغيل الخادم المحلي
npm run dev

# بناء الم��روع للإنتاج
npm run build

# معاينة البناء
npm run preview
```

### إعداد البيئة

1. انسخ ملف `.env.example` إلى `.env`
2. قم بتحديث متغيرات البيئة حسب الحاجة

```bash
cp .env.example .env
```
